\# Airline Database Project



\## Описание

База данных для управления авиаперевозками



\## Структура базы данных



\### Таблицы:

\- \*\*users\*\* - пользователи системы (администраторы, операторы)

\- \*\*aircrafts\*\* - самолеты

\- \*\*flights\*\* - рейсы

\- \*\*passengers\*\* - пассажиры

\- \*\*tickets\*\* - билеты

\- \*\*login\_history\*\* - история входов в систему



\## Схема данных

\- Один самолет может выполнять много рейсов

\- Один рейс может иметь много билетов

\- Один пассажир может иметь много билетов

\- Один пользователь может оформить много билетов

\- Один пользователь имеет много записей в истории входов



\## Тестовые данные

\- 3 пользователя (admin, operator1, operator2)

\- 3 самолета (Boeing 737, Airbus A320, Boeing 777)

\- 3 рейса (SU-100, SU-101, S7-250)

\- 3 пассажира (Иванов, Петрова, Сидоров)

\- 3 билета (15A, 15B, 1A)
